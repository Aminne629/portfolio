Code

Description

J'ai développé avec 2 coéquipiers un site pour la société Atos. Ce site est un site fictif, fait pour un projet scolaire. On y retrouve tout le nécessaire pour se renseigner sur la société Atos. Classification automatique de dépèches

Code

Description

Nous avons développé une méthode d'apprentissage automatique, sous-domaine de l'intelligence artificielle, qui permet à notre programme de classer une dépêche dans une catégorie en l'entraînant sur des dépêches. Pour chaque catégorie, on obtenait des mots représentatifs qui nous aidaient à déterminer la catégorie de la dépêche que l'algorithme devait classer.

Objectifs

  • Classer des dépèches
  • Apprendre à gérer les deadlines (temps très limité)
  • Coder sans indications au préalable des professeurs.

Travail en groupe

Équipe : 2 personnes

Durée : 5 jours

Méthodologie : Sprint

Ma contribution

  • Développement de l'algorithme
  • Tests et débogage

Technologies utilisées
